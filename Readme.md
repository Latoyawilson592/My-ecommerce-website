# NexShop - Modern E-Commerce Storefront

A lightning-fast, sleek, and fully responsive desk setup storefront. Built with pure vanilla JavaScript, dynamic layouts, CSS transitions, and Tailwind styling via CDN.

## Feature Set Included:
- **State Persistence:** LocalStorage tracks added cart items across refreshes.
- **Search Filtration:** Instant live query search across titles, specifications, and categories.
- **Dynamic Product Controls:** Filter items seamlessly by designated category tags.
- **Dynamic Pricing Sorters:** Order catalog lists in ascending or descending price ranges.
- **Adaptive Drawer Shopping Cart:** Add items, decrement/increment volumes, or remove line items in-drawer with updated totals.
- **Checkout Success Modal:** Simulated visual order success validation workflow.

## Hosting & Deployments via GitHub Pages:
1. Create a **new public repository** on GitHub.
2. Commit and push these 4 files directly to the `main` branch.
3. Open your repository on GitHub, navigate to **Settings** > **Pages**.
4. Set the Source to **Deploy from a branch**, choose **main** (or root), and click **Save**.
5. Your live site will deploy at `https://[your-username].github.io/[your-repo-name]/`.
